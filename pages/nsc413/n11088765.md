### 万科再提活下去 大陆三大房产公司举动引关注
------------------------

<p>
 【大纪元2019年03月04日讯】（大纪元记者骆亚、周慧心采访报导）近日
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%87%E7%A7%91.html">
  万科
 </a>
 再提活下去；
 <a href="http://www.epochtimes.com/gb/tag/%E7%A2%A7%E6%A1%82%E5%9B%AD.html">
  碧桂园
 </a>
 出现大规模裁员；而
 <a href="http://www.epochtimes.com/gb/tag/%E6%81%92%E5%A4%A7%E9%9B%86%E5%9B%A2.html">
  恒大集团
 </a>
 折价出售房产。大陆房地产三大巨头的举动，是否预示2019年房市会进入冷冬引关注。
</p>
<p class="p1">
 <span class="s1">
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%87%E7%A7%91.html">
   万科
  </a>
  的自媒体平台《万科周刊》2月25日刊登了董事会主席郁亮在集团内部的发言，再提万科要活下去，他说：“经济下行压力加大，‘L’型走势尚未探底，宏观形势并不乐观”，并表示，2019年要“收敛聚焦，巩固提升基本盘”，保证万科活下去、活得好、活得久。
 </span>
</p>
<p class="p1">
 <span class="s1">
  2月28日，澎湃新闻引述多个消息来源称，中国最大住宅开发商
  <a href="http://www.epochtimes.com/gb/tag/%E7%A2%A7%E6%A1%82%E5%9B%AD.html">
   碧桂园
  </a>
  的员工正受到“裁员”的困扰，有部门裁员幅度至50%。报导称此次裁员涉及范围从总部到区域。不过碧桂园在3月1日回应否认裁员，称公司是对组织架构进行优化调整。
 </span>
</p>
<p class="p1">
 <span class="s1">
  另外
  <a href="http://www.epochtimes.com/gb/tag/%E6%81%92%E5%A4%A7%E9%9B%86%E5%9B%A2.html">
   恒大集团
  </a>
  最近推出购房优惠，购房可以享有92折、95折、三万元购房券以及赠送家电等多重优惠。
 </span>
</p>
<p class="p1">
 <span class="s1">
  华尔街见闻报导，今年开年，房地产销售继续低迷态势。根据资讯统计机构克而瑞以及部分房企自己披露的数据，1月，排名前30的房企仅3家销售增速高于去年，碧桂园、恒大、万科等龙头企业降幅尤为明显，降幅都在50%以上。
 </span>
</p>
<p class="p1">
 <span class="s1">
  美国南卡罗莱纳大学艾肯商学院教授谢田认为，这反映
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%9B%BD%E6%88%BF%E5%9C%B0%E4%BA%A7.html">
   中国房地产
  </a>
  商对未来的展望不乐观。他对大纪元记者说：“这跟中国整个经济的下滑有关，现在中国房地产泡沫已经涨到如此庞大，中美贸易战继续僵持，让房地产业和其它行业一样面临更大的压力。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他表示，中共最害怕房地产业的倒闭潮会导致大量抛售，那样恐造成整个房地产市场的泡沫破灭，“
 </span>
 <span class="s1">
  它不敢让它破灭，它又有政治上的权利，它可以用国营企业贷款，国有银行的贷款来维持泡沫。”
 </span>
</p>
<p class="p1">
 他认为房地产绑架了中国的经济，地方政府官员、老百姓的钱都被捆绑在房地产市场里，一旦房地产市场泡沫破灭，中国的经济就会垮台。
</p>
<p class="p1">
 “房地产问题实际上与贷款问题联系在一起，中共更担心杠杆危机，就是借贷、贷款，债务市场的泡沫崩盘的问题；它担心产能过剩的问题；它也担心贸易战造成产业转移；再一个就是担心外汇储备减少，以及人民币汇率的崩溃。”
</p>
<p class="p2">
 <span class="s1">
  他说：“中国现在的经济局势就是按下葫芦起了瓢。哪个地方先松动，哪个地方就会先破灭，哪一个对它的政权有最大的威胁，它才优先去处理。”#
 </span>
</p>
<p class="p2">
 责任编辑：高静
</p>
