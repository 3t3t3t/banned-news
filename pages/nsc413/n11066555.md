### 黑龙江大庆79人被绑架 瞿延来被关3月多
------------------------

<p>
 【大纪元2019年02月24日讯】
 <span class="s2">
  2018
 </span>
 <span class="s1">
  年
 </span>
 <span class="s2">
  11
 </span>
 <span class="s1">
  月
 </span>
 <span class="s2">
  9
 </span>
 <span class="s1">
  日
 </span>
 <span class="s1">
  ，黑龙江大庆市发生一起大规模
  <a href="http://www.epochtimes.com/gb/tag/%E7%BB%91%E6%9E%B6.html">
   绑架
  </a>
  案。近日，有知情者向大纪元报料，当天警察按名单抓人，有
  <span class="s2">
   79
  </span>
  名
  <a href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </a>
  学员在同一天被
  <a href="http://www.epochtimes.com/gb/tag/%E7%BB%91%E6%9E%B6.html">
   绑架
  </a>
  。
 </span>
</p>
<p>
 此前，明慧网报导，11月9日，大庆市至少有61位
 <a href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
  法轮功
 </a>
 学员被警察绑架，多数被抄家，二十来人被送进大庆市看守所关押，十多人被送拘留所关押，十余人被骚扰或绑架未遂。
</p>
<p class="p1">
 <span class="s1">
  绑架案是由黑龙江省公安厅国保处副处长杨波、大庆市公安局国保支队长冯海波滥用职权，将信仰“真、善、忍”的法轮功学员诬蔑为
 </span>
 <span class="s2">
  “
 </span>
 <span class="s1">
  黑帮团伙
 </span>
 <span class="s2">
  ”
 </span>
 <span class="s1">
  ，纠集大批警力进行的一次违法违宪的行动。
 </span>
</p>
<p class="p1">
 <span class="s1">
  当地国保公安挥霍纳税人的血汗钱，长期对法轮功学员进行通讯（电话、手机、电报、微信）监控、电脑网络监控，采取跟踪、蹲坑等手段，指使大庆市区、县各公安分局警察按名单在
 </span>
 <span class="s2">
  11
 </span>
 <span class="s1">
  月
 </span>
 <span class="s2">
  9
 </span>
 <span class="s1">
  日同一天，实施群体绑架和骚扰，警察直接带领
 </span>
 <span class="s2">
  “
 </span>
 <span class="s1">
  开锁大王
 </span>
 <span class="s2">
  ”
 </span>
 <span class="s1">
  撬锁，入室抢劫。
 </span>
</p>
<p class="p1">
 <span class="s1">
  此外，杨波亲自到大庆负责
 </span>
 <span class="s2">
  “
 </span>
 <span class="s1">
  督办
 </span>
 <span class="s2">
  ”
 </span>
 <span class="s1">
  ，唆使推动迫害，同时搜集和罗织所谓的材料。大庆市各分局、部门抽人成立临时
 </span>
 <span class="s2">
  “
 </span>
 <span class="s1">
  专案组
 </span>
 <span class="s2">
  ”
 </span>
 <span class="s1">
  ，警察称之为
 </span>
 <span class="s2">
  “
 </span>
 <span class="s1">
  国保专案
 </span>
 <span class="s2">
  ”
 </span>
 <span class="s1">
  ，出动警车、特警车，多则十余辆，到大庆市看守所非法提审法轮功学员，并实施酷刑折磨，有的人已多次出现生命危险。
 </span>
</p>
<p class="p1">
 <span class="s1">
  同时，警察们上下互相串通，封锁一切消息。
 </span>
</p>
<p class="p1">
 <span class="s1">
  据不完全统计，目前已有
 </span>
 <span class="s2">
  11
 </span>
 <span class="s1">
  人被大庆市让胡路区检察院非法批捕。
 </span>
</p>
<p class="p1">
 <span class="s1">
  其中，居住在龙凤区的法轮功学员
  <a href="http://www.epochtimes.com/gb/tag/%E7%9E%BF%E5%BB%B6%E6%9D%A5.html">
   瞿延来
  </a>
  ，由于其户口在外地，原本没有在警察的
 </span>
 <span class="s2">
  79
 </span>
 <span class="s1">
  名抓捕名单之列。警察在监控、蹲坑监视其他法轮功学员的过程中，发现瞿与其他法轮功学员有往来，于是偷偷在
  <a href="http://www.epochtimes.com/gb/tag/%E7%9E%BF%E5%BB%B6%E6%9D%A5.html">
   瞿延来
  </a>
  的斯柯达轿车底部的排气管旁安装了摄像头，非法监控、跟踪了他三四
 </span>
 <span class="s1">
  天。
 </span>
</p>
<p class="p1">
 <span class="s2">
  11
 </span>

 </span>
 <span class="s1">

<p class="p1">

 </span>
</p>
<p class="p1">
 <span class="s1">
  警察在房间里控制住瞿延来的父母，再将他强行绑架到龙凤区东光公安分局。瞿延来家中遭到警察抢劫、一车私有财产被警察开车载走。
 </span>
</p>
