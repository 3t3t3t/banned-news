### 管控律师 拥护共产党成律师考核条件
------------------------

<p>
 【大纪元2019年04月06日讯】（大纪元记者骆亚、周慧心采访报导）中共司法部日前宣布，8月底前将在全国推行律师专业评价体系和评定机制。审核标准包括律师的政治表现，即是否拥护共产党领导。有律师认为这与
 <a href="http://www.epochtimes.com/gb/tag/%E4%BE%9D%E6%B3%95%E6%B2%BB%E5%9B%BD.html">
  依法治国
 </a>
 背道而驰。
</p>
<p class="p1">
 <span class="s1">
  这项律师审核机制从2017年3月开始在内蒙古、上海、安徽以及陕西试点。中共司法部称，试点效果“良好”，将推行到全国。
 </span>
</p>
<p class="p1">
 <span class="s1">
  据中共司法部网站消息，参评条件包括：政治表现、诚信状况、执业年限、执业能力。放在首位的“政治表现”明确规定，参评的律师应当拥护共产党领导，拥护“社会主义法治”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  而“诚信状况”则称，如果律师在参与评核前的5年内，受到党纪处分以及行政处罚等，都不能够参加评核。
 </span>
</p>
<p class="p1">
 <span class="s1">
  大陆
  <a href="http://www.epochtimes.com/gb/tag/%E4%BA%BA%E6%9D%83%E5%BE%8B%E5%B8%88.html">
   人权律师
  </a>
  程海对大纪元记者表示，80年代就有过这种评级，后
 </span>
 <span class="s1">
  <span class="s1">
   来废止了，因为“受不受认可是办案的质量，以及律师的执业能力，
  </span>
  <span class="s1">
   最后还是由社会认可，
  </span>
  <span class="s1">
   后来就没推行下去”。
  </span>
 </span>
</p>
<p class="p1">
 <span class="s1">
  他表示，中共要求律师服从党的领导，但是中共官员自己做的事情，跟其政策完全背道而驰，“共产党也要在法律下活动。党章规定，共产党员应该遵守法律。很多共产党员不守法，他们自己才是反党的。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“拥护共产党的领导是句空话，跟遵守法律是一个道理。党的领导其实就是听领导的，不是守法的概念。很多党员都是精神分裂，说一套做一套，不正常的人。所以如果说反党，他们自己才是反党。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  中国
  <a href="http://www.epochtimes.com/gb/tag/%E4%BA%BA%E6%9D%83%E5%BE%8B%E5%B8%88.html">
   人权律师
  </a>
  滕彪表示，在任何一个法治国家，都不可能把对党、对政府的忠诚度以及政治观点作为评价律师的标准。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他认为，律师是一个独立的法治职业，对于维护一个国家的法治精神起到非常重要的作用。“应该独立于政党，独立于政治意识形态，以维护宪法、维护法律和正义为天职。”他说。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他对大纪元记者说：“如果要求律师去拥护党的领导，必须要服从共产党的话，这是和律师制度的基本精神背道而驰的。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  滕彪认为中共从各个方面严防死守，害怕颜色革命，害怕和平演变，所以加强对律师意识形态的控制，是可以想像得到的。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “虽然维权律师只占律师群体很小的比例，但中共要对整体律师群体进行整肃，它不但限制维权律师，还进一步把思想、言论不可靠的人也列入整肃的队伍。”他说。
 </span>
</p>
<p class="p1">
 <span class="s1">
  近年来，民间人士被非法剥夺权利的事件越来越多，很多有良知的律师纷纷站出来，利用法律来帮助民众维权。然而这些被外界称为“人权律师”、“维权律师”的律师也同样遭到中共当局的迫害。
 </span>
</p>
<p class="p1">
 <span class="s1">
  2015年7月9日，中共在大陆境内展开一连串逮捕维权律师的行动，被称为“709事件”。据了解，“709事件”被逮捕下狱及被拘留、约谈、传唤、限制自由等遭遇的律师及相关人员有320多位。其中，备受外界关注的王全璋律师，目前仍生死不明。
 </span>
</p>
<p class="p1">
 北京维权律师余文生，因代理“709”律师案件，多次遭当局打压。他申请成立“智增律师事务所”时，收到当局的拒批通知，指他“多次发表反对党的领导、攻击社会主义法治的言论”，其行为不合乎“律师事务所管理办法”的规定等，所以申请不予许可。
</p>
<p class="p1">
 <span class="s1">
  而律师胡林政在为法轮功学员做了无罪辩护后，去年遭到司法局停止执业六个月的处罚。
 </span>
</p>
<p class="p1">
 <span class="s1">
  处罚书中说他违反了《律师执业管理办法》第二条，“律师应当把拥护中国共产党领导、拥护社会主义法治作为从业的基本要求。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  滕彪说：“他们（中共）认为，律师对代理的敏感案件进行无罪辩护，是对当局权威的挑战，尤其是很多人权律师提出信仰自由，提出宪法进行抗辩，在当局看来，就是对现有体制的挑战。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他认为，律师在中国一直没有取得独立的地位，被政府牢牢地控制。“中共对整个自由力量、对民间社会的全面控制已经越来越猖獗了，中国社会的矛盾会更加尖锐突出。用法律解决问题的渠道现在被完全堵死了，这对整个中国社会的稳定、健康发展是非常不利的。”他说。
 </span>
</p>
<p class="p2">
 <span class="s1">
  他说：“如果维权律师这个群体都没有了，或没办法正常工作的话，那所有群体都会受影响，除了最弱势的群体外，那些中产阶级，甚至包括一些官员、富商等等，他们也会成为受害者。”#
 </span>
</p>
