### 嘲笑加拿大总理陷麻烦 中共官媒帮倒忙
------------------------

<p>
 【大纪元2019年03月09日讯】（大纪元记者骆亚、周慧心采访报导）正在
 <a href="http://www.epochtimes.com/gb/tag/%E5%AD%9F%E6%99%9A%E8%88%9F.html">
  孟晚舟
 </a>
 引渡案喧嚣之时，加拿大总理
 <a href="http://www.epochtimes.com/gb/tag/%E7%89%B9%E9%B2%81%E5%A4%9A.html">
  特鲁多
 </a>
 陷“干预司法”事件中。中共官媒对此幸灾乐祸。有评论认为该事件让特鲁多更不敢冒险释放
 <a href="http://www.epochtimes.com/gb/tag/%E5%AD%9F%E6%99%9A%E8%88%9F.html">
  孟晚舟
 </a>
 ，反而会变得更加强硬，中共搞不清楚状况。
</p>
<p class="p1">
 <span class="s1">
  加拿大前检察总长兼司法部长王州迪（Jody Wilson-Raybould）2月27日在国会司法委员会做证，称在去年9月～12月期间，包括
  <a href="http://www.epochtimes.com/gb/tag/%E7%89%B9%E9%B2%81%E5%A4%9A.html">
   特鲁多
  </a>
  在内的11名政府高官持续向她施加不恰当压力，让其帮助被控腐败和欺诈的SNC-Lavalin公司免受刑事起诉。此案让特鲁多陷入“干预司法”的指控。
 </span>
</p>
<p class="p1">
 <span class="s1">
  中共多个官媒都对此案进行了追踪、幸灾乐祸的报导。
  <br/>
 </span>
</p>
<p class="p2">
 <span class="s1">
  对此，大陆独立媒体分析人士吴特表示，正是因为加拿大司法透明，这个案件才会出现。“中共的党媒、外宣一直在孟晚舟事情上攻击加拿大政府，对于特鲁多被指干涉司法的丑闻，中共党媒跟着幸灾乐祸。”
 </span>
</p>
<p class="p2">
 <span class="s1">
  他对大纪元记者说：“美方提出犯罪证据和引渡要求，加拿大依法抓人并进入引渡程序，这是加拿大司法系统的既定程序。如果特鲁多的丑闻是真的，也不代表会停止引渡孟晚舟的程序。另外，如果特鲁多插手司法要求放孟晚舟，那才是真正的破坏司法独立。”
 </span>
</p>
<p class="p2">
 <span class="s1">
  对于中共媒体称特鲁多可能迫于压力会下台之说，吴特认为中共的行为很滑稽。“如果特鲁多因这个丑闻下台，最终得益的会是对
  <a href="http://www.epochtimes.com/gb/tag/%E5%8D%8E%E4%B8%BA.html">
   华为
  </a>
  乃至中共更强硬的加拿大保守党，这对中共来讲并不是好事，他们为此幸灾乐祸，甚至推波助澜，其实是搬起石头砸自己的脚。”
 </span>
</p>
<p class="p2">
 <span class="s1">
  他说：“我不知道中共外交部还有党媒外宣那些人是真不懂加拿大政治还是装的不懂，既定司法程序被说成是‘执意引渡’，特鲁多被更强硬的右翼攻击，被中共说成‘中国或成最大的赢家’，看着中共觉得挺滑稽的。”
 </span>
</p>
<p class="p2">
 <span class="s1">
  他表示，因为加拿大司法独立，允许当事人申辩，所以引渡之事估计会拖很久，但是既定程序肯定会走下去，中共是阻止不了的。“如果他们还继续玩绑架加拿大人和盲目攻击加拿大现政府那一套，反倒会陷孟晚舟于不利。”
 </span>
</p>
<h4 class="p2">
 中共频频帮倒忙
</h4>
<p class="p2">
 <span class="s1">
  自从孟晚舟被抓后，中共通过媒体、外交部、公安等多方面对加拿大施压，企图阻止孟晚舟被引渡。但其多个行为被认为是“帮倒忙”。
 </span>
</p>
<p class="p3">
 <span class="s1">
  在
  <a href="http://www.epochtimes.com/gb/tag/%E5%8D%8E%E4%B8%BA.html">
   华为
  </a>
  极力撇清与中共关系之时，中共官方似乎生怕别人不知道它与华为的“亲密关系”。
 </span>
</p>
<p class="p3">
 2月28日，包括《纽约时报》和《华盛顿邮报》在内的美国媒体记者纷纷“晒”出一份来自华为的采访邀请。但这些邀请信竟然都是经由中共驻美大使馆发出的。
</p>
<p class="p3">
 <span class="s1">
  除了中共媒体、外交部之外，中共警方为了给加拿大施压，抓捕了加拿大公民。3月1日加拿大司法部批准并正式启动引渡孟晚舟的司法程序后，中共中央政法委官方新闻网站长安网在3月4日发文称，加拿大前外交官
  <a href="http://www.epochtimes.com/gb/tag/%E5%BA%B7%E6%98%8E%E5%87%AF.html">
   康明凯
  </a>
  自2017年以来，通过中国境内的关系人，窃取、刺探中国敏感信息和情报。而加拿大商人斯帕弗是康明凯的重要情报关系人，向康明凯提供情报。
 </span>
</p>
<p class="p4">
 <span class="s1">
  对此，中国时政评论家文昭表示，公众得知案件消息，不是通过媒体，就是执法部门直接开新闻发布会向公众告知，但这个消息不是来自于执法部门，而是由中共政法委，这个党的机构释放出来。
 </span>
 <span class="s1">
  他说：“由中共中央政法委网站公告，这事本身就很讽刺。”
 </span>
</p>
<p class="p1">
 加拿大总理特鲁多3月4日在夏洛特敦（Charlottetown）出席一个新闻活动时，对此做出了回应。特鲁多说政府非常关注中方的的这种举动，并表示这不会影响加拿大政府继续孟晚舟案的引渡程序。
</p>
<p class="p4">
 经济学博士、时事评论员李松筠认为，特鲁多陷入麻烦后，为了避嫌，他更不可能冒“干预司法”的风险去释放孟晚舟。她说：“中共搞不清楚状况，还嘲笑特鲁多，这只会让特鲁多更加强硬，真是越帮越忙。”#
</p>
