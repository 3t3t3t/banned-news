### 两会“木偶戏”结束 李克强记者会说法被质疑
------------------------

<p>
 【大纪元2019年03月15日讯】（大纪元记者骆亚、周慧心采访报导）2019年中国的“
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%A4%E4%BC%9A.html">
  两会
 </a>
 ”今天结束，国务院总理李克强在人大会堂回答中外记者提问时，否认政府要求企业监听它国活动，并强调“现在没有，将来也决不会有”。但学者认为：过去、现在、将来都会有，政府的政策本身就迫使企业这么做。
</p>
<p class="p1">
 <span class="s1">
  3月15日的记者会上，彭博新闻社记者提问，中共政府是否会迫使其企业监视它国？李克强回应：“我不知道指的是监听政府的还是公民个人的，反正有一条，这样做不符合中国法律，也不是中国行事的方法，现在不会有，将来也决不会有。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  有网民分析说，李克强只否认现在和将来，并没有提及过去，是否有意回避过去监听的行为。北京时局观察员华颇对大纪元记者说：“我的看法是过去、现在、将来都会，政府的政策本身就迫使企业这么做。”
 </span>
</p>
<p class="p2">
 <span class="s1">
  中共的“
  <a href="http://www.epochtimes.com/gb/tag/%E6%83%85%E6%8A%A5%E6%B3%95.html">
   情报法
  </a>
  ”第七条规定：任何组织和公民都应当依法支持、协助和配合国家情报工作；第十二条规定：国家情报工作机构可以按照国家有关规定，与有关个人和组织建立合作关系，委托开展相关工作；第十四条规定：国家情报工作机构依法开展情报工作，可以要求有关机关、组织和公民提供必要的支持、协助和配合。
 </span>
</p>
<p class="p2">
 <span class="s1">
  而“
  <a href="http://www.epochtimes.com/gb/tag/%E5%8F%8D%E6%81%90%E6%80%96%E4%B8%BB%E4%B9%89%E6%B3%95.html">
   反恐怖主义法
  </a>
  ”第十八条：“电信业务经营者、互联网服务提供者应当为公安机关、国家安全机关依法进行防范、调查恐怖活动提供技术接口和解密等技术支持和协助。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  中国问题专家、悉尼科技大学冯崇义教授表示，中共当然不会承认自己是贼，“他们这种答复没有任何的惊讶，很平常的一个表白。为了中共的利益，他对外只能这么说。他不可能承认。”
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  “
  <a href="http://www.epochtimes.com/gb/tag/%E5%A4%96%E5%95%86%E6%8A%95%E8%B5%84%E6%B3%95.html">
   外商投资法
  </a>
  ”规范政府行为
 </span>
</h4>
<p class="p1">
 <span class="s1">
  这次中共人大通过了“
  <a href="http://www.epochtimes.com/gb/tag/%E5%A4%96%E5%95%86%E6%8A%95%E8%B5%84%E6%B3%95.html">
   外商投资法
  </a>
  ”，该法将于2020年1月1日正式实施。
 </span>
 <span class="s1">
  记者会上，亚洲新闻台记者提问表示，有舆论担心“外商投资法”中部分法律条款模糊，可能会给当局灵活掌握的空间。
 </span>
</p>
<p class="p1">
 <span class="s1">
  李克强回答说，“这个法是要用法律的手段更好保护外商投资，吸引外商投资。这个法律也可以说是规范政府行为的，要政府依法行政，而且政府要根据这个法律的精神出台一系列法规、文件，保护外商权益⋯⋯”
 </span>
</p>
<p class="p1">
 <span class="s1">
  华颇认为这个法律很难实施，“外企到中国设立企业，就要雇用中国人进行管理和经营、生产、制造，在这种情况下很难做到技术的保密。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“这种事情政府是不会出面的，都由相应的企业对企业，所以虽然美国提出了知识产权的保护等，但是很难从根本上转变这种状况。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  据美国之音3月15日报导，在法律和国家安全网站“法律战”（Lawfare）最近的一篇文章中，华盛顿的咨询顾问和分析师洛伊强调法案中有条文规定，外国企业不应“损害国家安全或公共利益”，影响国家安全的企业将受到审查。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“这些条文实质上是在给予国家——中国共产党——完全自由干涉广泛的投资活动，提醒外国投资者，他们最好还是避免对那些可能被认为是政治敏感或政治威胁领域的任何投资。”
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  这次
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%A4%E4%BC%9A.html">
   两会
  </a>
  被称“木偶戏”
 </span>
</h4>
<p class="p1">
 <span class="s1">
  冯崇义表示，这次的两会无聊至极。“一个个噤若寒蝉，不敢说话，所有的人都不能‘高级黑’，也不能‘低级红’，所有人都担心稍微说出不合适的话，就会遭殃。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“很不正常。这个会原来还会做些表演，现在连装模作样都不做了，完全就像木偶戏一样。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他表示，这次中共完全不做任何的掩饰，“大兵直接进到会场里，把代表们都镇住，整个政治已经完全堕落、退化成无聊的木偶戏。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  华颇也认为这次两会平淡无奇，没有什么亮点。“只是像台湾的57岁姑娘惺惺作态，令人作噁。”另外他表示今天李克强的记者会他也失去看的兴趣。“李克强的记者招待会，限制的非常死，一味地背书，并没有自己真实的意愿。”#
 </span>
</p>
