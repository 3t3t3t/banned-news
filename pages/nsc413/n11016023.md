### 供销合作社回头 离“凭票买粮”有多远？
------------------------

<p>
 【大纪元2019年02月01日讯】（大纪元记者骆亚、周慧心采访报导）随着原证监会主席刘士余赴中华全国供销合作总社履新，中共官媒纷纷高调介绍供销合作总社。有学者认为，在
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E6%88%98.html">
  中美贸易战
 </a>
 的压力下，中共最终可能会倒退回
 <a href="http://www.epochtimes.com/gb/tag/%E8%AE%A1%E5%88%92%E7%BB%8F%E6%B5%8E.html">
  计划经济
 </a>
 。
</p>
<p class="p1">
 <span class="s1">
  <a href="http://www.epochtimes.com/gb/tag/%E4%BE%9B%E9%94%80%E5%90%88%E4%BD%9C%E7%A4%BE.html">
   供销合作社
  </a>
  是为农村服务的合作经济组织，是
  <a href="http://www.epochtimes.com/gb/tag/%E8%AE%A1%E5%88%92%E7%BB%8F%E6%B5%8E.html">
   计划经济
  </a>
  的产物。其前身可以追溯到1949年11月成立的中央合作事业管理局。1954年，更名为中华全国供销合作总社，建立了全国统一的供销合作社系统。但在改革开放之后，“供销社”一度淡出人们的视野。
 </span>
</p>
<p class="p1">
 <span class="s1">
  据新华社报导称，
  <a href="http://www.epochtimes.com/gb/tag/%E4%BE%9B%E9%94%80%E5%90%88%E4%BD%9C%E7%A4%BE.html">
   供销合作社
  </a>
  系统自2013年以来，在中国恢复重建基层社一万多家，使基层供销社总数超过三万家，乡镇覆盖率从2012年的56%提高到2018年的95%。
 </span>
</p>
<p class="p1">
 <span class="s1">
  中美贸易谈判关键时刻，在川普要求中共进行
  <a href="http://www.epochtimes.com/gb/tag/%E7%BB%93%E6%9E%84%E6%80%A7%E6%94%B9%E9%9D%A9.html">
   结构性改革
  </a>
  的压力下，官媒突然高调宣传供销合作社，难道计划经济又要回头？
 </span>
</p>
<p class="p1">
 <span class="s1">
  美国南卡罗莱纳大学艾肯商学院教授谢田认为，这正好迎合了中共政治上倒退的社会形势。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “它实际上跟结构改革背道而驰，（中华全国供销合作总社）这是中共国务院管理的，相当于部级单位，是国家控制的一个机构，跟美国的要求恰恰相反。如果中共不得不走向计划经济的旧模式的话，它只能采用合作社的方式，来开展它的供销活动。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他对大纪元记者说：“现在中共政治上向左，经济上在
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E6%88%98.html">
   中美贸易战
  </a>
  下，很多人说要恢复计划经济体制，打击私有企业。供销合作总社覆盖了基层、乡村、城镇的95%，如果中共在贸易战下不得不收紧，倒退回计划经济，商品配送合作社就是它最重要的一环。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  悉尼科技大学的中国问题专家冯崇义认为，实际上现在中共是没路可走了，他对大纪元记者说：“它现在把供销合作社再做回来，好像把一个僵尸，已经埋了几十年了，又挖出来，成为它救经济的方法之一。因为没招了，就想这些歪招，很搞笑。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  而经济学博士李松筠则表示，中共意识到
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%9B%BD%E7%BB%8F%E6%B5%8E.html">
   中国经济
  </a>
  面临危机，可能会回到五六十年代物资极度匮乏的状况，她对大纪元记者说：“它（中共）不得不统一收购和分派物资，说不定还会发粮票了。”
 </span>
</p>
<p class="p1">
 粮票是1955年～1993年间中国计划经济时期的粮食配给凭证，在统购统销背景下，城镇居民的个人生存完全依赖于这些票证，尤其是在发生大饥荒的1960年代初。除粮票外，还有肉票、油票、鸡蛋票、糖票、布票、煤票、肥皂票等。当时以粮票为主的票证，则完全进入流通领域，甚至具有了货币的功能。
</p>
<p class="p1">
 谢田表示，从多个方面可以看出，现在国内金融和经济的每一部分都岌岌可危。“2018年最后两三个月，实际上很多出口企业抓紧出口，抢在关税提高之前，让中国的顺差创了纪录。但是2019年这个抢点的效果没有了，贸易肯定会大大地受挫。”
</p>
<p class="p1">
 <span class="s1">
  他认为，2019年中国产业链外移的情况会继续，中国失业问题会加剧。“进入到今年7月份，高校800万大学生毕业的时候，中国的就业问题会更加严重。现在很多公司在裁员，几百万公司注销，几千万人失业，再加上新的大学毕业生，很多公司开始冻结、停止招工，所以就业问题会非常地严峻。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“贸易顺差没有的话，将导致外汇储备减少，债务问题很快发酵，又会触发房地产泡沫的问题⋯⋯这个基本上是中共防不胜防的。”#
 </span>
</p>
