### 理悟：也说“压舱石”
------------------------

<p>
 【大纪元2019年02月25日讯】
</p>
<p>
 风急浪高月黑天，旋涡连连船欲翻。
 <br/>
 中共忙搬压舱石，可惜底漏稳船难。
 <br/>
 一号文件话
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E5%86%9C.html">
  三农
 </a>
 ，年年许愿篇连篇。
 <br/>
 喜歌唱罢七十载，农民依然苦年年。
 <br/>
 农民朴实听党喝，
 <a href="http://www.epochtimes.com/gb/tag/%E8%8A%B1%E8%A8%80%E5%B7%A7%E8%AF%AD%E8%A2%AB%E5%85%9A%E9%AA%97.html">
  花言巧语被党骗
 </a>
 。
 <br/>
 打倒土豪分田地，农民骗上井冈山。
 <br/>
 革命成功田到手，好景不长政策变。
 <br/>
 一夜之间公社化，土地归公攥空拳。
 <br/>
 流血卖命换来啥？二等公民苦不堪。
 <br/>
 农民户口如枷锁，祖祖辈辈翻身难。
 <br/>
 三年饥荒党作祸，饿死农民四千万。
 <br/>
 官商勾结开发潮，农民失地被强迁。
 <br/>
 外出打工挣俩钱，党妈寒冬驱低端。
 <br/>
 有苦想与党妈诉，
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%8A%E8%AE%BF.html">
  上访
 </a>
 路尽牢里关。
 <br/>
 邪党用你举头顶，用过一脚踹溜扁。
 <br/>
 土地永远不私有，农民翻身永远难。
 <br/>
 小康美景一画饼，年年新画惹人馋。
 <br/>
 是甜是香啥滋味？空咽吐沫口水干。
 <br/>
 上当百回终觉醒，认清好歹不听骗。
 <br/>
 邪党今遇亡命事，又对农民打算盘。
 <br/>
 你说
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E5%86%9C.html">
  三农
 </a>
 压舱石，我说石压快沉船。
 <br/>
 三农不是避风港，农民心凉听你唤？
 <br/>
 轻忽悠，休行骗，今日农民非往年。
 <br/>
 且听耳畔雷雨急，民心如水覆红船。
</p>
