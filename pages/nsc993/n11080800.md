### 关乐：柏林墙守卫案审判
------------------------

<p>
 【大纪元2019年03月01日讯】
</p>
<p>
 一九八九严冬夜，
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E6%9F%8F%E6%9E%97%E5%A2%99.html">
  柏林墙
 </a>
 头寒气烈。
 <br/>
 卄岁高定克里斯，
 <br/>
 两小伙同把墙越。
 <br/>
 东德守卫亨里奇，
 <br/>
 举枪瞄准连发射。
 <br/>
 击中克里斯前胸，
 <br/>
 造成最末遇难者。
 <br/>
 高定脚踝也中弹，
 <br/>
 一死一伤酿罪过。
 <br/>
 九个月后墙推倒，
 <br/>
 德国统一消红色。
 <br/>
 一九九一年九月，
 <br/>
 卫兵被审
 <a href="http://www.epochtimes.com/gb/tag/%E9%A3%9F%E6%81%B6%E6%9E%9C.html">
  食恶果
 </a>
 。
 <br/>
 亨里奇判三年半，
 <br/>
 假释不得牢实坐。
 <br/>
 律师辩称奉命令，
 <br/>
 罪不在己没选择。
 <br/>
 法官摇头不认可，
 <br/>
 明知无辜还开火。
 <br/>
 不从命令会获罪，
 <br/>
 打不准确无罪责。
 <br/>
 枪口抬高一厘米，
 <br/>
 这一主权自掌握。
 <br/>
 良心义务在此刻，
 <br/>
 主动承担系应做。
 <br/>
 故事简单乃经典，
 <br/>
 报应临头没法躲。
