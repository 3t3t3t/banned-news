### 中立：跪活
------------------------

<p>
 【大纪元2019年04月04日讯】
</p>
<p>
 火球煎熬着神州，看似酷热的白昼；
 <br/>
 却是黑暗的长夜，分分秒秒如一年；
 <br/>
 具备行走的动物；多数无知的动物；
 <br/>
 一切
 <a href="http://www.epochtimes.com/gb/tag/%E5%AD%98%E6%B4%BB%E7%9A%84%E6%84%8F%E4%B9%89.html">
  存活的意义
 </a>
 ，全部锁定在活着；
 <br/>
 多少个日日夜夜，跪下去才能活着；
 <br/>
 艰难中
 <a href="http://www.epochtimes.com/gb/tag/%E8%B7%AA%E6%B1%82%E6%B4%BB%E7%9D%80.html">
  跪求活着
 </a>
 ，得到的都是
 <a href="http://www.epochtimes.com/gb/tag/%E6%81%90%E6%83%A7.html">
  恐惧
 </a>
 。
 <br/>
 毫无底线的沉默，无情地抹灭灵魂；
 <br/>
 天地圣坛的真谛，在垃圾堆里埋藏；
 <br/>
 那些无知咆哮着，古圣言都是糟粕；
 <br/>
 从有记忆开始时，直到盖棺定论时；
 <br/>
 唯一焦热的火球，才是万物的必须。
 <br/>
 宇宙万物有定数，无论花言再巧语；
 <br/>
 无知带来的恶果，必将被悲惨埋葬；
 <br/>
 直到最终才明白，救命稻草是圣言。
</p>
