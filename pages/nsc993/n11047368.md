### 理悟：看红朝
------------------------

<p>
 【大纪元2019年02月15日讯】
</p>
<p>
 抬望眼，看
 <a href="http://www.epochtimes.com/gb/tag/%E7%BA%A2%E6%9C%9D.html">
  红朝
 </a>
 ，
 <br/>
 大好河山一团糟。
 <br/>
 美丽祖国不复在，
 <br/>
 满目疮痍雾霾罩。
 <br/>
 民不聊生如草芥，
 <br/>
 毒水毒食毒疫苗。
 <br/>
 陡生多少癌症村，
 <br/>
 千家万户悲泪抛。
 <br/>
 看
 <a href="http://www.epochtimes.com/gb/tag/%E7%BA%A2%E6%9C%9D.html">
  红朝
 </a>
 ，乱红朝，
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1%E6%B5%81%E6%B0%93.html">
  中共流氓
 </a>
 走歪道。
 <br/>
 招来川普贸易战，
 <br/>
 经济崩盘叹萧条。
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E9%82%AA%E5%85%9A%E6%83%B9%E7%A5%B8.html">
  邪党惹祸
 </a>
 百姓扛，
 <br/>
 又要逼民勒裤腰。
 <br/>
 昔日毛魔造饥荒，
 <br/>
 赤地千里倒饿殍。
 <br/>
 看红朝，弃红朝，
 <br/>
 歌舞升平强颜笑。
 <br/>
 八哥鹦鹉齐学舌，
 <br/>
 党妈满脸胭脂膏。
 <br/>
 台上台下红毒漫，
 <br/>
 愚弄百姓麻醉药。
 <br/>
 只是国人已渐醒，
 <br/>
 火眼金睛识红妖。
 <br/>
 看红朝，倒红朝，
 <br/>
 百姓只愿祖国好。
 <br/>
 剿灭共匪清红毒，
 <br/>
 祖国母亲直起腰。
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E6%8E%80%E7%BF%BB%E6%9A%B4%E6%94%BF.html">
  掀翻暴政
 </a>
 迎民主，
 <br/>
 从整山河看娇娆。
 <br/>
 神传五千续华夏，
 <br/>
 万众高歌祖国好。
</p>
