### 理悟：莫误上天赐机缘
------------------------

<p>
 【大纪元2019年01月12日讯】
 <br/>
 血腥黑幕七十年，万恶中共台上演。
 <br/>
 一路杀来血旗舞，整死饿死八千万。
 <br/>
 毛魔运动连番搞，神州处处冤魂冤。
 <br/>
 土改三反又五反，反右文革花样翻。
 <br/>
 阶级斗争无神论，打天骂地人性变。
 <br/>
 一颗红心献给党，爹娘祖宗不值钱。
 <br/>
 夫妻反目揭丈夫，父子成仇划界线。
 <br/>
 为保自己断亲情，违心下手斗字先。
 <br/>
 儿子贴父大字报，斗父脚踹肋骨断。(注1)
 <br/>
 也有小小少年郎，头戴尖帽父牵连。
 <br/>
 堂堂神州绞肉机，是非颠倒人伦乱。
 <br/>
 人不务正穷革命，身上无衣锅没饭。
 <br/>
 三面红旗呼啦啦，饿殍遍野炊烟断。
 <br/>
 邪党依然伟光正，毛魔作死民遭难。
 <br/>
 黑白二猫蹿上台，国门大开向钱看。
 <br/>
 一阵急风天地昏，官红二代撒欢干。
 <br/>
 国有资产充己有，跑马圈地欲无边。
 <br/>
 忧国忧民学子泪，泣泪化血溅天安。
 <br/>
 邓猫个小贼胆大，六四屠城史无前。
 <br/>
 放话杀人二十万，保党稳坐二十年。
 <br/>
 邪党性邪不容正，耀邦紫阳靠边站。
 <br/>
 跳出奇坏江蟾蜍，六四踏血坐金銮。
 <br/>
 二奸二假人皆知，贪财好色百病全。
 <br/>
 出卖国土知多少？面积四十多台湾。
 <br/>
 治国无能祸国能，闷声发财举国乱。
 <br/>
 无官不贪党之风，黄赌毒黑大泛滥。
 <br/>
 捞钱行业归谁有？党官家家有财源。
 <br/>
 落马高官查露馅，百亿千亿贪不鲜。
 <br/>
 苦了百姓腰筋断，几代辛劳穷依然。
 <br/>
 上学住房愁无路，大病等死没有钱。
 <br/>
 江家儿孙齐上阵，贪欲无底垒金山。
 <br/>
 坏事做绝红伴黑，任意横行上海滩。
 <br/>
 江大蛤蟆邪毒狠，打压正信手段残。
 <br/>
 打死法徒算自杀，就地火化灭尸源。
 <br/>
 名誉搞臭经济断，肉体消灭无底线。
 <br/>
 酷刑百种人发指，活摘器官换金钱。
 <br/>
 长子遭报得肾癌，活人配型找肾源。
 <br/>
 换肾三次杀五人，百姓命贱任草菅。
 <br/>
 活摘形成产业链，党政军警各争先。
 <br/>
 白衣天使刽子手，医院倒成屠宰间。
 <br/>
 滔天罪恶未收手，中共本性不曾变。
 <br/>
 活摘酷刑依然在，一年杀人五六万。
 <br/>
 国际追查步步紧，铁证如山逃罪难。
 <br/>
 嘴硬心虚头冒汗，遮遮掩掩躲几天？
 <br/>
 民悬水火千般苦，红魔当道无人权。
 <br/>
 一切财富归党有，百姓两手攥空拳。
 <br/>
 跟党造反打土豪，分得土地两天半。
 <br/>
 到头土地归共产，白当农民没地权。
 <br/>
 党要用地少补偿，敢不同意强拆迁。
 <br/>
 据理抗争党镇压，见血棍棒橡皮弹。
 <br/>
 声名狼藉国际臭，耍尽流氓抢偷骗。
 <br/>
 惹得川普横眉对，一怒发起贸易战。
 <br/>
 刚一接火败阵来，两腿哆嗦心发颤。
 <br/>
 邪不压正节节退，弯腰屈膝嘴服软。
 <br/>
 已是末路日难熬，经济崩溃喷火山。
 <br/>
 企业倒闭千千万，外企退出厂搬迁。
 <br/>
 瞬间失业大潮起，千万家庭要吃饭。
 <br/>
 火上房子等水救，真服假服耍心眼。
 <br/>
 要想中共真改好，日头西出海水干。
 <br/>
 邪灵附体骨子坏，权贵集团根难断。
 <br/>
 当政打虎半载美，虎头江曾狂依然。
 <br/>
 百姓期望化泡影，权利内斗一窝乱。
 <br/>
 空谈改革假改革，一党专政手遮天。
 <br/>
 依法治国法无存，民主踩在脚下边。
 <br/>
 这种邪党留何用？天灭中共是必然。
 <br/>
 邪不胜正天理在，物极必反谁能拦？
 <br/>
 山雨欲来风满楼，天怒人怨掀红船。
 <br/>
 四面楚歌中共危，八方围剿烽火燃。
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E4%B9%9D%E8%AF%84.html">
  九评
 </a>
 九剑斩赤龙，邪党挣扎根已断。
 <br/>
 引发
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
  三退
 </a>
 潮头涌，三亿两千四百万。
 <br/>
 人人选择弃邪党，解体中共无形间。
 <br/>
 别抱希望被蒙蔽，党拉陪葬冤不冤？(注2)
 <br/>
 谁还迷糊快快醒，快做
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
  三退
 </a>
 是正选。
 <br/>
 高低贵贱一个样，莫误上天赐机缘。
</p>
