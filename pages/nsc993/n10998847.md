### 理悟：年关滋味
------------------------

<p>
 【大纪元2019年01月24日讯】
</p>
<p>
 过年啦，喜洋洋，有喜有悲不一样。
 <br/>
 党官富豪阖家乐，奢侈无度当福享。
 <br/>
 电视歌舞庆升平，穷汉梦里奔小康。
 <br/>
 肩扛党国千张嘴，酸甜苦辣一年尝。
 <br/>
 小儿嚷嚷买鞭炮，丫头盼身花衣裳。
 <br/>
 可惜爹爹兜里瘪，手中无钱心白忙。
 <br/>
 年年打工挣俩钱，今年生变赶回乡。
 <br/>
 企业倒闭活难找，低端被弃马路旁。
 <br/>
 平地强征补偿少，山地贫脊不打粮。
 <br/>
 一步一坎日子紧，展望明天眼茫茫。
 <br/>
 别说高官多贪腐，小小村官也张狂。
 <br/>
 拉票送俺一袋面，当上村官眼朝上。
 <br/>
 卖山卖地发横财，
 <a href="http://www.epochtimes.com/gb/tag/%E7%9B%98%E5%89%A5%E7%99%BE%E5%A7%93.html">
  盘剥百姓
 </a>
 喝稀汤。
 <br/>
 扶贫低保谁受益？姑舅两姨三大娘。
 <br/>
 四顾茫然皆黑夜，何时天亮出太阳？
 <br/>
 还是东街张嫂对：三退解体中共党。
 <br/>
 共党不倒天不晴，永远阴天没太阳。
 <br/>
 再听中共瞎忽悠，辈辈穷苦无希望。
</p>
