### 鲁泰：一代意大利一足球路
------------------------

<p>
 【大纪元2019年03月27日讯】
</p>
<p>
 体育
 <a href="http://www.epochtimes.com/gb/tag/%E6%96%87%E9%9D%A9.html">
  文革
 </a>
 ，贺龙难昌。
 <br/>
 南昌朱德，叶挺省港。
 <br/>
 省港跑路，天蟾沪冈。
 <br/>
 沪冈谍匪，唯唯苏傍。
 <br/>
 苏傍蒙内，通日窃帮。
 <br/>
 窃帮非正，内斗叛党。
 <br/>
 叛党平反，少奇榜样。
 <br/>
 榜样雷锋，神祖轰棒。
 <br/>
 轰棒未远，足球俱伤。
 <br/>
 俱伤一孩，意国视殃。
 <br/>
 视殃大陆，球迷甲宕。
 <br/>
 甲宕贪腐，股市球殇。
 <br/>
 球殇一带，一路新疆。
 <br/>
 新疆集中，器官血浆。
 <br/>
 血浆供蟾，一贪永康。
 <br/>
 永康薄谷，苏家屯绑。
 <br/>
 屯绑胡习，法徒难当。
 <br/>
 难当活摘，罪大荡荡。
 <br/>
 荡荡天门，佛恩浩荡。
 <br/>
 浩荡佛恩，
 <a href="http://www.epochtimes.com/gb/tag/%E7%A5%9E%E9%9F%B5.html">
  神韵
 </a>
 场场。
 <br/>
 场场爆满，秀美阳刚。
 <br/>
 阳刚阳刚，马衰复阳。
 <br/>
 复阳
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
  三退
 </a>
 ，观潮钱塘。
 <br/>
 钱塘四海，黄河长江。
 <br/>
 长江溯源，昆仑青藏。
</p>
