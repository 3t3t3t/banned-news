### 委内瑞拉事件冲击中国 民心思变 中共惊恐
------------------------

<p>
 【大纪元2019年01月25日讯】（大纪元记者骆亚、周慧心采访报导）
 <a href="http://www.epochtimes.com/gb/tag/%E5%A7%94%E5%86%85%E7%91%9E%E6%8B%89.html">
  委内瑞拉
 </a>
 事件再次打响了世界与社会主义之间的“战争”。大陆民众欢呼“民心思变”，中共当局则谨防“
 <a href="http://www.epochtimes.com/gb/tag/%E9%A2%9C%E8%89%B2%E9%9D%A9%E5%91%BD.html">
  颜色革命
 </a>
 ”。大陆民主人士在接受大纪元采访时，首度被提醒“你正在接听的电话XXXX，为境外来电，请注意防范风险。”
</p>
<p class="p1">
 <span class="s1">
  反对派领袖瓜伊多1月23日宣誓就任
  <a href="http://www.epochtimes.com/gb/tag/%E5%A7%94%E5%86%85%E7%91%9E%E6%8B%89.html">
   委内瑞拉
  </a>
  临时总统后不久，美国立即宣布承认瓜伊多的临时总统身份。并表示不排除对委内瑞拉采取军事干预手段。
 </span>
</p>
<p class="p1">
 <span class="s1">
  随后，巴西、哥伦比亚、秘鲁、加拿大等十多个国家相继承认瓜伊多的临时总统身份。到目前为止，除玻利维亚、古巴、尼加拉瓜和墨西哥外，美洲大多数国家不再与
  <a href="http://www.epochtimes.com/gb/tag/%E9%A9%AC%E6%9D%9C%E7%BD%97.html">
   马杜罗
  </a>
  政府建立外交关系。而
 </span>
 <span class="s1">
  中共外交部发言人连续两天发言宣称反对外部干预委内瑞拉事务。
 </span>
</p>
<p class="p1">
 <span class="s1">
  曾经富有的委内瑞拉，在
  <a href="http://www.epochtimes.com/gb/tag/%E9%A9%AC%E6%9D%9C%E7%BD%97.html">
   马杜罗
  </a>
  的统治下变成赤贫，大陆前法官沈良庆表示，这说明社会主义道路行不通。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他对大纪元记者说：“马杜罗是左翼总统，反映了委内瑞拉选民本身有社会主义倾向，结果社会主义道路造成很大的经济危机，这说明社会主义道路是行不通的，最后老百姓用抗议示威的方式把他抛弃。”
 </span>
</p>
<p class="p2">
 <span class="s1">
  2014年委内瑞拉人均GDP1.65万美元，是南美最富裕的国家，该国曾实行全民免费医疗，从幼儿园到大学的教育全部免费，低收入家庭由政府免费提供食品和生活必需品，曾被中国媒体称为“人间天堂”。
 </span>
</p>
<p class="p2">
 <span class="s1">
  可是，自从2013年4月14日，马杜罗赢得选举成为总统后，委内瑞拉便陷入近代史上最严重的经济危机：面粉、牛奶、肉品、豆类等主要食品短缺。
 </span>
</p>
<p class="p2">
 <span class="s1">
  该国通货膨胀全球最高，根据国际货币基金组织（IMF）去年10月的报告，委内瑞拉的年通货膨胀率在年底前飙升至1,370,000%。现在该国上街购物，钱不是拿来数的，而是称重量。
 </span>
</p>
<p class="p2">
 <span class="s1">
  由于食物极度匮乏，该国到处发生打砸超市的行为，为防止骚乱，武装部队被迫荷枪实弹进驻各个超市，维持食物的计划配给。
 </span>
</p>
<h4 class="p2">
 <span class="s1">
  中国民众关注 因人心思变
 </span>
</h4>
<p class="p2">
 <span class="s1">
  对于委内瑞拉变天的事件，在西方国家引发的关注度远没有在中国人圈子里引发的关注高，不少知道真实消息的中国人都欢欣鼓舞。沈良庆认为这正是中共当局最担心的——人心思变。
 </span>
</p>
<p class="p2">
 <span class="s1">
  “实际上人们关注某个事件，跟他们自身是有关的。”他解释说：“中国人特别关注委内瑞拉，一片欢呼声，这反映了一种情绪，就是中国人自己有这种政治需求，就希望这种变天。这也是中共当局最害怕的。”
 </span>
</p>
<p class="p2">
 有网民甚至调侃说，这两年中共的老朋友都哪儿去了？巴基斯坦总理谢里夫全家被捕；马来西亚前总理纳吉布因贪腐被捕； 2019年刚开头，委内瑞拉总统马杜罗岌岌可危⋯⋯
</p>
<p class="p1">
 沈良庆对此表示，虽然这是网上的调侃，但也确实是事实。“这说明中共流氓集团力挺的都是小流氓，这些小流氓政权本身也有很多的问题，不是中共挺它挺死的，而是因为流氓政权在哪里都是不受欢迎的，它自己垮台。”
</p>
<h4 class="p2">
 接受境外来电被提醒“防范风险”
</h4>
<p class="p1">
 <span class="s1">
  对于中共外交部频繁发言且措辞强硬，沈良庆认为，除了中共和委内瑞拉的关系比较密切，经济上有很多的合作和援助之外，更重要的是中共的危机感很强。
 </span>
</p>
<p class="p2">
 <span class="s1">
  “比如前几天习近平在中央党校召开的专题研讨班上的讲话就非常明显，防范风险、防范灰犀牛和黑天鹅。”
 </span>
</p>
<p class="p2">
 <span class="s1">
  他解释说：“灰犀牛就是整个危机的开始，黑天鹅就是小概率事件。他危机感非常强，政权风雨飘摇，政治上要严格把关，他肯定不希望像委内瑞拉变天那样事情发生。”
 </span>
</p>
<p class="p2">
 <span class="s1">
  他表示，最近媒体报导说微软的搜索引擎Bing被关了，“本来这个搜索引擎使用的用户很少，也不重要，至少我此前都没听说过，这都反应了当局的紧张。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  中共现在非常恐惧，沈良庆告诉记者，他在跟记者通话过程中，他的电话屏幕上显示：“中国移动短信，尊敬的客户你好，你正在接听的电话XXXX，为境外来电，请注意防范风险。”他说，这是他第一次看到这种现象。#
 </span>
</p>
