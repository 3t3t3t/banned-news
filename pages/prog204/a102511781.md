### 卫星图曝光 中共在西藏至少建三集中营
------------------------

<div class="post_content">
 <p>
  【新唐人2019年02月14日讯】继续关心中国的消息。尽管中共矢口否认
  <a href="https://www.ntdtv.com/gb/新疆.htm">
   新疆
  </a>
  或
  <a href="https://www.ntdtv.com/gb/西藏.htm">
   西藏
  </a>
  地区“
  <a href="https://www.ntdtv.com/gb/再教育营.htm">
   再教育营
  </a>
  ”的存在，但印度媒体引用
  <a href="https://www.ntdtv.com/gb/卫星图片.htm">
   卫星图片
  </a>
  ，直接戳破
  <a href="https://www.ntdtv.com/gb/中共谎言.htm">
   中共谎言
  </a>
  。
 </p>
 <p>
  印度媒体，The Print网站在2018年8月，曾以
  <a href="https://www.ntdtv.com/gb/卫星图片.htm">
   卫星图片
  </a>
  证实，中共在
  <a href="https://www.ntdtv.com/gb/新疆.htm">
   新疆
  </a>
  兴建了多座“
  <a href="https://www.ntdtv.com/gb/再教育营.htm">
   再教育营
  </a>
  ”，来迫害维族人。如今，The Print 再度引用卫星图片说，指出
  <a href="https://www.ntdtv.com/gb/西藏.htm">
   西藏
  </a>
  地区，目前至少有三座“再教育营”正在兴建。西藏地区的宗教、语言和文化，也面临中共的严厉监控与打压。
 </p>
