### 4月19日维权动态
------------------------

<div class="post_content" itemprop="articleBody">
 <p>
  【新唐人2019年04月19日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 </p>
 <p>
  <strong>
   河北访民北京寄信被殴打关押
  </strong>
  <br/>
  河北省廊坊市固安县访民张建侠，4月15号在北京西单寄信时被北京警方抓捕并送到马家楼黑监狱关押，之后被固安县接访人员接回，期间遭到暴力殴打。16号张建侠被关进固安县看守所，家属没有收到任何通知。
 </p>

  <br/>
  “709案”受害律师
  <a href="https://www.ntdtv.com/gb/王全璋.htm">
   王全璋
  </a>
  的妻子
  <a href="https://www.ntdtv.com/gb/李文足.htm">
   李文足
  </a>
  ，4月19号与朋友一起，前往天津高院询问王全璋案上诉情况，结果遭到法警的故意刁难，不准进入法院。王全璋被中共抓捕1370多天，家属一直不知道他的真实状况，无奈之下，她们在天津高院门前举牌抗议。
 </p>

   <a href="https://www.ntdtv.com/gb/郭泉.htm">
    郭泉
   </a>
   被拘留
  </strong>
  <br/>
  南京著名政治犯
  <a href="https://www.ntdtv.com/gb/郭泉.htm">
   郭泉
  </a>
  4月18号再次被抓，原因是他在“郭泉语录”中, 声援要求政府公开江苏响水爆炸信息被刑拘的曹建山。维权网报导说，警方的初步处理是拘留10天。目前, 郭泉被关在南京市拘留所。
 </p>

  <a href="https://www.ntdtv.com/gb/任照.htm">
   任照
  </a>
