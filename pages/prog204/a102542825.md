### 【禁闻】3月27日维权动态
------------------------

<div class="post_content" itemprop="articleBody">
 <p>
  【新唐人2019年03月27日讯】
  <br/>
  下面我们来关注一下中国大陆各地民众维权的情况。
  <br/>
  邯郸天津铁厂数千
  <a href="https://www.ntdtv.com/gb/职工维权.htm">
   职工维权
  </a>
  遭镇压
  <br/>
  位于河北邯郸市涉县的天津铁厂，3月23号，爆发大型职工抗议活动，数千名工人聚集在神山广场，然后从广场一路游行到厂区，沿途高呼“罢工、罢工”、“我们要生存”等口号。当局调动数千名警察镇压，有职工被抓、被打伤。
 </p>
 <p>
  内蒙古农民丁贵雄因“
  <a href="https://www.ntdtv.com/gb/翻墙.htm">
   翻墙
  </a>
  ”被传唤
  <br/>
  内蒙古巴彦淖尔盟五原县塔尔湖镇农民丁贵雄，3月22号遭县公安局警察上门检查电脑和手机，并带到国保大队办公室讯问，警告他不得再“
  <a href="https://www.ntdtv.com/gb/翻墙.htm">
   翻墙
  </a>
  ”（也就是突破网络封锁）看被中共封锁的境外网站信息。丁贵雄被扣押到深夜11点半才被释放。
 </p>
 <p>
  709家属天津中院索要
  <a href="https://www.ntdtv.com/gb/王全璋.htm">
   王全璋
  </a>
  判决书
  <br/>
  3月27号，709家属李文足、刘二敏、王峭岭、原珊珊带着两个年幼的孩子，前往天津第二中级法院索要
  <a href="https://www.ntdtv.com/gb/王全璋.htm">
   王全璋
  </a>
  律师的判决书，但主审法官周虹、林崑，和法院的负责人都避而不见，李文足等人最后愤怒的在大厅中高喊“周虹，林崑，给我判决书”，但仍然没有人理睬她们。王全璋5岁多的儿子泉泉质问：“有本事抓我爸爸，没有本事出来。”
 </p>
 <p>
  沈阳
  <a href="https://www.ntdtv.com/gb/访民.htm">
   访民
  </a>
  失踪女儿发公开信寻人
  <br/>
  辽宁沈阳沈北新区
  <a href="https://www.ntdtv.com/gb/访民.htm">
   访民
  </a>
  郭佩丽，3月14号在北京火车站候车回家时，被北京警察带走交给地方警察, 之后一直与外界失联，她的儿子到当地警方查询，被告知无登记记录，家属也一直没有收到拘留通知书等。3月26号，郭佩丽的女儿在网上发表致北京市公安局局长王小洪的公开信，询问母亲下落。
 </p>
